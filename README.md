# 🎬 boungcing_imgs

Welcome to a creative and playful version of the classic DVD bouncing logo screensaver! ✅

## 🚀 What it does
- Loads all images from `imgs/` (any supported format by Pygame)
- Resizes each image to **20x20 px**
- Spawns one bouncing sprite per image with random velocity
- Simulates DVD-style edge collisions on full screen
- Uses a **black background** for maximum contrast

## 🖼️ Visual behavior
- Every sprite bounces in x/y with real-time collision response
- Unlimited images are supported, each is independent
- Perfect for a dynamic screensaver demonstration

## 🛑 Exit instructions
- A floating prompt appears every few seconds at the bottom center:
  - `Press ESC to exit the screensaver`
- Text fades in / persists / fades out softly (4 seconds total display)

## 🎯 Run it
```bash
python bouncings.py
```

## 💡 Tip
For best effect, add fun icons or logos to the `imgs/` folder and watch them dance across the screen!
